#  Grubbs
 
An 2D pixel Platformer Game made in Unity.
 
**How it works:** GamePlay is Simple as Normal Platformer Game, here you will just crash out alot that is one thing.
 
---
 
## Controls
- Space: Jump
- A: Walk Left
- D: Walk Right
- LMB: Attack
 
---
 
## Features
- Wall Jump
- Health
- Death
- Attack
- Enemies
- Traps
- Traps with AI
- Option to change the Music output
- Option to change the Sound Effects output
---
Note:- Crazyy Skill set required to complete
